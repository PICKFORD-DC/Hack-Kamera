## Installing In Termux 

```bash
apt update && apt upgrade
apt install git -y
apt install curl -y
apt install php -y
apt install unzip -y
apt install wget -y
apt install whiptail -y
git clone https://github.com/PICKFORD-DC/Hack-Kamera 
cd Hack-Kamera
bash setup 
```
